# 浏览器兼容性

支持以下版本的浏览器：

- Chrome 版本大于等于 111
- Edge 版本大于等于 111
- Firefox 版本大于等于 114
- Safari 版本大于等于 16.4
- 市场份额大于等于 0.5% 的浏览器版本
- Firefox 的长期支持版本（ESR）
- 每个浏览器的最近两个版本

最后，排除超过 24 个月没有官方支持或更新的浏览器。
