# Browser Compatibility

::: info Info

This documentation is AI-generated. You can help improve it by submitting an [Issue](https://github.com/HowieHz/halo-theme-higan-hz/issues/new).

:::

Supports the following browser versions:

- Chrome version 111 or higher
- Edge version 111 or higher
- Firefox version 114 or higher
- Safari version 16.4 or higher
- Browser versions with market share of 0.5% or higher
- Firefox Extended Support Release (ESR)
- The last two versions of each browser

Finally, browsers that have not received official support or updates for more than 24 months are excluded.
