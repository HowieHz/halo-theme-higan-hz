## 通过 iconify 获取图标

此方法获取的 SVG 格式为 data URI，百分号编码：`url("data:image/svg+xml,%3Csvg...%3E")`

前往[iconify](https://icon-sets.iconify.design/)，选择任意图标集后选择你要的图标。

点击图标后，底部会弹出一展示框，左边一列选择 `CSS`，顶栏选择 `URL` 即可看到 `url("……")` 格式的图标信息。

点击 `Copy to Clipboard` 按钮将此信息复制到剪切板。

找到主题对应位置，粘贴即可。
