# Theme Toggle Button Component

## Description

This component renders the shared theme toggle button used by post-style headers and the site header.

## Usage

Head Content (for including necessary scripts/styles)

```html
<th:block th:insert="~{components/theme-toggle-button/template :: head}"></th:block>
```

Body Content

```html
<th:block th:insert="~{components/theme-toggle-button/template :: body}"></th:block>
```
