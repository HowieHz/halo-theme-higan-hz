# Nav Page-Like-Post-Style Component

## Description

This component renders the floating side navigation used by page-like-post-style pages.

## Usage

Head Content (for including necessary scripts/styles)

```html
<th:block th:insert="~{components/nav-page-like-post-style/template :: head}"></th:block>
```

Navbar Content

```html
<th:block th:insert="~{components/nav-page-like-post-style/template :: body}"></th:block>
```
