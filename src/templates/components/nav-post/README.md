# Nav Post Component

## Description

This component renders the floating side navigation used by post pages.

## Usage

Head Content (for including necessary scripts/styles)

```html
<th:block th:insert="~{components/nav-post/template :: head}"></th:block>
```

Navbar Content

```html
<th:block th:insert="~{components/nav-post/template :: body}"></th:block>
```
