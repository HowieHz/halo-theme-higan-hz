# Footer Nav Page-Like-Post-Style Component

## Description

This component renders the bottom floating action bar used by page-like-post-style pages.

## Usage

Head Content (for including necessary scripts/styles)

```html
<th:block th:insert="~{components/footer-nav-page-like-post-style/template :: head}"></th:block>
```

Body Content

```html
<th:block th:insert="~{components/footer-nav-page-like-post-style/template :: body}"></th:block>
```
